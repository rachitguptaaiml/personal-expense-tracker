# Personal Expense Tracker
A simple web application to track your expenses with a bar chart.

## Features
- Add new expense with date, category and amount
- View all expenses in a table
- Display a bar chart of total expenses by category

## Setup/Run Instructions
1. Clone the repository: `git clone https://github.com/your-username/personal-expense-tracker.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run the application: `python main.py`
